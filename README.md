# Windows-AppWipe
Remember to wipe your apps!   Lists all Win 10/11 apps and provides easy removal and cleanup.

---

## 📱 Features
- ✅ Color-coded app removal recommendations for Windows 10 / 11
- ✅ Self elevating (runs as admin)
- ✅ Fast & Easy bulk "remove all" option (safe to remove apps only)
- ✅ Protects/hides essential apps to prevent issues
- ✅ Custom friendly names (Xbox App, Widgets, etc.)
- ✅ Multi-user removal (existing + new profiles)
- ✅ Provisioned package cleanup (no auto reinstalls)
- ✅ Nothing special needed. Just raw powershell.

---

## 🚀 Quick Start
👉 Download latest release: https://github.com/OutlawCecil/Windows-AppWipe/releases  
👉 UnZip the PowerShell script  
👉 Right click -> Run with PowerShell (double click may work for some)  
👉 Enter the # of the app you'd like to remove and press enter.  

---

## 📊 App Categories
🟢 **GREEN** = Recommend keeping  
⚪ **WHITE** = Optional apps  
🔴 **RED** = Bloatware for most users  
🟣 **PURPLE** = Bulk remove ALL reds
